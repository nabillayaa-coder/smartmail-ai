# SmartMail AI

SmartMail AI is a full-stack AI-powered email management application built with React, TypeScript, FastAPI, and Python.

## Features

- Modern responsive email dashboard
- Inbox, Important, Spam, AI Priority, and Sent categories
- Compose and send emails
- Search and filter emails
- AI-powered email analysis
- Spam detection
- Priority classification
- AI-generated summaries
- Suggested AI replies
- Full frontend ↔ backend communication

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Python
- FastAPI
